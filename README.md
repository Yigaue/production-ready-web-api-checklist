# Production-Ready Web & API Checklist

A comprehensive, framework-agnostic checklist for building **secure, scalable, and maintainable** web and API applications.

This repository is intended to be used as:
- A pre-production checklist
- A pull-request review guide
- A team engineering baseline
- A learning reference for best practices

## Who This Is For
- Backend engineers
- Full-stack engineers
- Tech leads & reviewers
- Startup teams shipping fast
- Enterprise teams standardising quality

## How To Use
- Copy individual checklist files into your project
- Use the PR checklist template in GitHub
- Fork and customise for your organisation
- Use as a training reference for new engineers

## Checklist Structure
Each item follows the format:
> **Action** — *Reason*

Example:
> Add a middleware to set `X-Frame-Options: DENY` — prevents clickjacking attacks by blocking iframe embedding.
